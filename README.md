# MyTravis
[![Build Status](https://travis-ci.org/lquixada/my-travis.png?branch=master)](https://travis-ci.org/lquixada/my-travis)

Monitor your projects builds from TravisCI within Chrome.

## Development

### Installing Tools

```bash
$ npm install
```

### Tests

```bash
$ npm test
```
